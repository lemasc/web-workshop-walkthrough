# 01 - Think in Responsive

## What is responsive design?

ในการออกแบบเว็บไซต์ในปัจจุบัน เราจำเป็นต้องคำนึงถึงการแสดงผลในหลาย ๆ อุปกรณ์มากกว่าสมัยก่อน ไม่ว่าจะเป็น
บนมือถือ แท็บเล็ต ไอแพด หรือกระทั่งบนคอมพิวเตอร์เองปัจจุบันก็มีการแสดงผลในหลายขนาด ดังนั้นเราต้องมั่นใจว่าเว็บเรา
**สามารถแสดงผลได้บนทุกอุปกรณ์ (Responsive)** นั่นเอง

![responsive](https://admin.12grids.com/uploads/blogs/original_cover_images/future-of-responsive-web-design-12grids.jpg)

## Traditional Design : Table

มาทำความเข้าใจการแสดงผลเว็บไซต์ในสมัยก่อน แล้วเราจะเข้าใจการออกแบบในสมัยนี้มากยิ่งขึ้น

เว็บไซต์ในสมัยก่อนแสดงบนจอคอมพิวเตอร์ที่มีลักษณะเป็น**แนวนอน** บนเว็บไซต์ส่วนมากก็จะมีองค์ประกอบไม่กี่อย่าง เช่น

- **Navigation Bar** อยู่ด้านบนสุดของหน้า ประกอบด้วยลิงก์เป็นแถว ๆ ให้กดไปยังหน้าต่าง ๆ
- **Content** แสดงเนื้อหาหลักของหน้านั้น ๆ ซึ่งในส่วน Content ก็อาจจะประกอบด้วยแถบเมนูทางด้านซ้าย และเนื้อหาแสดงคู่กันในด้านขวา (และในเนื้อหาก็อาจจะมีการแบ่งแถวย่อย ๆ อีก)
- **Footer** แสดงส่วนท้ายของเว็บ Credit ต่าง ๆ เป็นต้น

![](https://media.cheggcdn.com/media/e1a/e1a2eece-111c-4687-a72c-0d94eca93250/phpK9X5bo)

สมัยก่อนจึงมีแนวคิดในการ **ออกแบบเว็บไซต์ด้วยตาราง (Table)** ซึ่งอาศัยการคิดออกแบบหน้าเว็บเป็นแถว ๆ และคอลัมน์นั่นเอง

> [!TIP]
>
> ลองปรับแต่ละบรรทัดใน CSS เพื่อดูว่าผลลัพธ์จะแตกต่างกันยังไง โดยกดคีย์ลัด `Ctrl + /` เพื่อซ่อน (Comment) และเลิกซ่อน (Uncomment)
> หรือกดแก้แต่ละบรรทัดดื้อ ๆ เลยก็ได้
